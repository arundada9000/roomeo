# Changelog

All notable changes to Roomeo will be documented in this file.

## [0.1.0] — 2026-05-31

### Added

- Map-first room exploration with Leaflet + PostGIS
- Property/unit management for landlords
- Admin dashboard with analytics and moderation
- Better Auth integration (email/password)
- Role-based access control (Admin, Landlord, User)
- Favorites, inquiries, reviews
- Room-type PNG icons with 3 variant sets
- PWA support with offline fallback
- Blog, contact, newsletter, careers pages
- PostCSS v4 + TailwindCSS v4 styling
- Framer Motion animations throughout
- Zustand state management for map UI
- Server Components + Client Components architecture
- Prisma 7 ORM with PostgreSQL/PostGIS

### Tech

- Next.js 16.2.6 · React 19.2.4 · TypeScript 5
- Prisma 7.8.0 · PostgreSQL 16+ / PostGIS
- TailwindCSS 4 · shadcn/ui · Base UI
- Leaflet 1.9.4 · React Leaflet 5
- Better Auth 1.6.11
- Framer Motion 12.40.0 · Lucide React 1.16.0
- Zustand 5.0.13 · TanStack Query 5.100.14
