You are a senior staff-level software architect, product designer, UX engineer, and performance-focused fullstack engineer.

Your task is to build a production-grade modern room/flat finder platform using Next.js ecosystem with enterprise-grade architecture, clean scalable code, excellent UX, mobile-first design, geospatial intelligence, PWA support, and high performance.

====================================================
PROJECT OVERVIEW
====================================================

The platform is an advanced room/flat finder system where:

1. Admins manage the entire platform
2. Landlords upload/manage rooms and flats
3. Users search nearby available rooms/flats using maps and advanced filtering

This is NOT a generic real-estate website.
The platform must feel:
- instant
- mobile-first
- map-first
- modern
- minimal
- highly responsive
- trustworthy
- spatially intelligent

The core experience is:
“Find nearby available rooms/flats quickly using live location + smart filters.”

The app should feel like a mix of:
- Airbnb
- Uber Maps
- Google Maps
- modern SaaS dashboard
- high-end mobile application
